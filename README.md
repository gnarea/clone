# ghost-writing

A [Claude Code](https://claude.ai/claude-code) plugin that teaches Claude how to write on my behalf — covering tone, British English conventions, formatting, and context-aware style for formal, semi-formal, and informal writing.

## Installation

From within Claude Code, add this repo as a marketplace and install the plugin:

```
/plugin marketplace add gnarea/clone
/plugin install ghost-writing@gnarea
```

Repeat on each machine where you use Claude Code.

## Usage

The `ghost-writing` skill is not user-invocable. Claude applies it automatically when drafting or editing text on your behalf, selecting the appropriate style (formal, semi-formal, or informal) based on context.
